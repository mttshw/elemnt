Menu

You can add vertical or horizontal menus. They accept a elements or button elements as children. Or you can use an elemnt-menu-item component to give you slots at the start or end to use for icons.

Home About Services Contact 

<elemnt-menu orientation="horizontal">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Services</a>
    <a href="#">Contact</a>
</elemnt-menu>
Home About Contact 

<elemnt-menu orientation="vertical">
    <a href="#">Home</a>
    <a href="#">About</a>
    <button>Services</button>
    <a href="#">Contact</a>
</elemnt-menu>

Default alignment is to the left but you can change that.

Home About Services Contact 

<elemnt-menu orientation="horizontal" alignment="center">
    <a href="#">Home</a>               
    <a href="#">About</a>
    <a href="#">Services</a>
    <a href="#">Contact</a>
</elemnt-menu>
Home About Contact 

<elemnt-menu orientation="vertical" alignment="center">
    <a href="#">Home</a>               
    <a href="#">About</a>
    <button>Services</button>
    <a href="#">Contact</a>     
</elemnt-menu>

This example aslo has a disabled menu item

Home About Services Contact 

<elemnt-menu orientation="horizontal" alignment="right">
    <a href="#">Home</a>               
    <a href="#">About</a>
    <a href="#" disabled>Services</a>   
    <a href="#">Contact</a>
</elemnt-menu>
Home About Contact 

<elemnt-menu orientation="vertical" alignment="right">
    <a href="#">Home</a>               
    <a href="#">About</a>           
    <button>Services</button>
    <a href="#">Contact</a>     
</elemnt-menu>

By default menus stretch to fit their container. But you can restrict them to just the width of their content.

Home About Services Contact 

<elemnt-menu orientation="horizontal" width="content">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Services</a>
    <a href="#">Contact</a>
</elemnt-menu>
Home About Services Contact 

<elemnt-menu orientation="vertical" width="content">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Services</a>
    <a href="#">Contact</a>
</elemnt-menu>
